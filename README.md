# ML

Репозиторий с практическими реализациями ключевых алгоритмов машинного обучения и интеллектуального анализа данных. Все проекты реализованы в формате **Google Colab**

## Содержание проекта


### 1. Интеллектуальный анализ данных (Data Mining)

* **Ассоциативные правила:** Поиск скрытых закономерностей в транзакционных данных (алгоритмы типа Apriori или FP-Growth).
* **Кластеризация:** * `K-Means`: Классический метод разбиения на  кластеров, `STING` (Statistical Information Grid): Сеточный алгоритм кластеризации для работы с большими наборами пространственных данных.


### 2. Прогнозирование и Регрессия

* **Анализ временных рядов:** Исследование динамики данных, проверка на стационарность и построение прогнозов.
* **Логистическая регрессия:** Базовая модель классификации для оценки вероятностей событий.

### 3. Продвинутые методы ML

* **Классификация с CatBoost:** Использование градиентного бустинга над решающими деревьями от Яндекса. Оптимизировано для работы с категориальными признаками.
* **Нечеткая логика (Fuzzy:** Моделирование неопределенности и создание систем логического вывода
* **Reinforcement Learning:** Обучение агентов принятию решений в интерактивной среде.
* **Снижение размерности и визуализация многомерных данных:** Применение алгоритмов t-SNE и UMAP для проецирования сложных высокоразмерных данных на плоскость.
